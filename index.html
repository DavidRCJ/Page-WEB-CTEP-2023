Abstract

In order to understand the different stages of the development of a project through the V model, we made the implementation of a Door & Window Control Module for a car with the Door Locking and Window Controlfunctions.

Introduction

Through the Continental Talent Entry Program CTEP 2023, we acquire new knowledge in different disciplines and had the opportunity to implement a Window-Door Control Module integrating all the modules of the program and be able to perform a workrole at Continental.

Methodology

The development of the project follows the V model levels, starting with a high-level definition of the requirements, next stage Software Details Desing, moreover,configuration of Architecture with Rhapsody-Tresos-Cessar City follow the layers Autosar, next stage develop code, this code development in AURIX Development Studio based on board KIT_AURIX_TC275_ARD_SB. the last stage is develop test case unit, integrate, To verify the correct functionality of requirements.

Results

The Doors-Window Control Module functions were implemented correctly, achieving the safety goals required such as locking doors at high speeds and the use of an Antipinch protection in windows, complying with the proposed test cases.

Conclusion

Throughout the program we learned a lot about the development of projects at Continental, key technical elements and soft skills that we found very useful and hope to apply in the most efficient way for the development of our upcoming projects. Thaks Continental Thaks CTEP
